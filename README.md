# 🔐 RFID Solenoid Door Lock System

A Digital Logic Design project based on an **RFID-enabled Solenoid Door Lock System** using **Arduino UNO** and RFID technology for secure access control.  
The system authenticates users through RFID cards/tags and unlocks the door upon successful verification.

---

## ✨ Features

- ✅ RFID-based authentication system
- ✅ Contactless access control
- ✅ Automatic door locking and unlocking
- ✅ Visual status indication using LEDs
- ✅ Audible feedback using buzzer
- ✅ Secure and user-friendly design
- ✅ Arduino-controlled locking mechanism

---

## 🛠 Components Used

- Arduino UNO
- RFID Reader and RFID Tags
- Solenoid Lock
- Relay Module
- LEDs (Red and Green)
- Buzzer
- Breadboard
- Jumper Wires
- Battery
- USB Cable

---

## ⚙️ System Working

1. User scans an RFID card/tag
2. RFID reader sends data to Arduino UNO
3. Arduino verifies the RFID card
4. If authentication is successful:
   - Green LED turns ON
   - Buzzer gives feedback
   - Solenoid lock unlocks the door
5. Unauthorized access keeps the door locked
